# MPU6050连接示意图

## 硬件连接图

```
+-------------+                  +----------------+
|   MPU6050   |                  |    STM32F4     |
|             |                  |                |
|         VCC |-----------------→| 3.3V           |
|         GND |-----------------→| GND            |
|         SCL |-----------------→| PB8 (SCL)      |
|         SDA |←---------------→| PB9 (SDA)      |
|         INT |- - - - - - - - →| 未连接(可选)    |
|             |                  |                |
+-------------+                  +----------------+

注：I2C通信线SCL和SDA均需要外接上拉电阻(4.7kΩ)连接到3.3V
```

## MPU6050模块引脚图

```
+------------------+
|    MPU6050模块   |
|                  |
|  VCC  GND  SCL  SDA  XDA  XCL  AD0  INT |
+--+----+----+----+----+----+----+----+---+
   |    |    |    |    |    |    |    |
   |    |    |    |    |    |    |    |
 3.3V  GND   |    |   NC   NC   GND  NC/GPIO
          PB8(SCL) PB9(SDA)
          
注：
- XDA/XCL: 用于连接外部传感器，本项目未使用
- AD0: I2C地址选择，接地时地址为0x68，接高电平时地址为0x69
- INT: 中断输出，可选连接
```

## I2C总线上拉电阻连接

```
     3.3V
       |
       |
      +++
      | | 4.7kΩ上拉电阻
      +++
       |
       +-----+---------------+
       |     |               |
SCL ---+     |               |
             |               |
      +++    |               |
      | | 4.7kΩ上拉电阻      |
      +++                    |
       |                     |
       |                     |
SDA ---+---------------------+
```

## 注意事项

1. I2C总线上拉电阻值推荐为4.7kΩ，过大或过小都会影响通信质量
2. 连线尽量短，减少干扰
3. VCC和GND之间可以并联0.1μF的去耦电容
4. 确保MPU6050模块安装牢固，避免振动影响测量精度
