# 1. 功能

```
1. 可以通过LCD1602显示环境数据
2. 通过DHT11、MQ-2实时检测环境数据
3. 当环境数据异常时，报警
4. 使用LED代替执行机构
5. 可以通过蓝牙 远程调控 阈值

以上为必须实现的基本功能
其他的自由发挥，功能越多分数越高
```
